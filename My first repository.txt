Мій перший репозиторій

1. Git - це система керування версіями.
2. Існує три статуси файлів : UNTRACKED, UNMODIFIED та MODIFIED
3. Команда add додає файл до репозиторію
4. Команда status перевіряє, чи додані файли
5. Команда commit потрібна для фіксації стану сховища в певний час моменту
6. Команда init ініціалізує Git
7. В коментаріях комітів потрібно писати те, що змінилося у файлі
8. Список комітів можна переглянути за допомою команди git log
9. Команда checkout дозволяє пересуватися між гілками git
